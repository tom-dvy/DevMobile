1 - Placer un GameObject en enfant du prefab et le nommer "StartPoint"

--- IMPORTANT --- => Placer le StartPoint en 0, 0, 0

2 - Faire le segment de piste comme on le souhaite

3 - Placer un GameObject l'a où le segment s'arrête et le nommer "Endpoint"

4 - Placer le script "TrackSegment" sur le parent et y définir le type du segment créé
